## Error analysis

The module [**error_analysis**](error_analysis_documentation.md) containes a range of analysis methods, including statistical analysis and regressions, enabling you to visualize the results and derive valuable insights from your data. It provides an easy way to check statistics, explore correlations, and make predictions using regressions. See the [examples](error_analysis_examples.md) about the module usage.
