# Data Analysis Overview

This package is a comprehensive solution for data query, analysis and visualization. It is designed to work with data from your simulations that you set and ran in the cloud [remotely](https://citros.io/doc/docs_tutorials#working-with-citros-cli---online) or through [web](https://citros.io/). With the extensive features of **citros_data_analysis** package, you can quickly and easily extract valuable insights from your data.

**citros_data_analysis** is a Python package that enables you to query and analyze data generated by running simulations in the CITROS cloud. This package can be used either from the citros.io Python notebook located under each [repository's notebook](https://citros.io/doc/docs/repos/repository/repo_sc_code#directory-notebooks) or from a local kernel Python environment such as Jupyter, VSC, etc., after installing the [CITROS](https://pypi.org/project/citros/) package and logging in.
